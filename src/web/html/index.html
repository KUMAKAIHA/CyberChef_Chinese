        X 功能如何工作？

        CyberChef 具有上下文帮助功能。只需将光标悬停在您想要了解更多信息的功能上，然后按键盘上的 F1 即可获得有关它的信息。现在尝试将光标悬停在此文本上并按 F1！


        我可以使用 CyberChef 做哪些事情？

        CyberChef 中有 数百个 操作，可让您轻松执行简单和复杂的任务。以下是一些示例：

        • 解码 Base64 编码的字符串
        • 将日期和时间转换为不同的时区
        • 解析 Teredo IPv6 地址
        • 从 hexdump 转换数据，然后解压缩
        • 解密和反汇编 shellcode
        • 将多个时间戳显示为完整日期
        • 对不同类型的数据执行不同的操作
        • 使用输入的部分内容作为操作的参数

        我可以从文件直接加载输入吗？

        是的！只需将文件拖到输入框上并放下即可。

        CyberChef 可以处理最大约 2GB 的文件（取决于您的浏览器），但是，某些操作可能需要很长时间才能对这么多数据运行。

        如果输出大于某个阈值（默认 1MiB），它将作为可供下载的文件呈现给您。如果需要检查文件片段，可以在输出中查看它们。


        如何一次对多个输入运行操作 X？

        也许您有 10 个要解析的时间戳或 16 个都具有相同密钥的编码字符串。

        “Fork”操作（在“流程控制”类别中找到）按行拆分输入，并分别对每一行运行所有后续操作。然后，每个输出都显示在单独的行上。可以更改这些分隔符，因此如果您的输入以逗号分隔，则可以将拆分分隔符更改为逗号。

        单击此处 查看示例。


        “Magic”操作如何工作？

        “Magic”操作使用多种方法来检测编码数据以及可用于理解数据的操作。可以在 此处 找到这些方法的技术描述。

        什么
        
                                
        一个简单、直观的 Web 应用程序，用于分析和解码数据，而无需处理复杂的工具或编程语言。CyberChef 鼓励技术和非技术人员探索数据格式、加密和压缩。

        

                                
        为什么
        
                                
        在现代世界中，数字数据以各种形状、大小和格式出现 – CyberChef 帮助在一个易于使用的平台上理解这些数据。

        


                                
        如何
        
                                
        该界面在设计时以简洁为核心。复杂的技术现在变得像拖放一样简单。简单的功能可以组合起来构建“Recipe”，可能会产生复杂的分析，这些分析可以与其他用户共享并与他们的输入一起使用。
        
                                
        对于那些习惯编写代码的人来说，CyberChef 是一种快速高效的方式来为问题原型化解决方案，一旦证明有效，就可以编写脚本。

        


                                
        谁
        
                                
        预计 CyberChef 将对网络安全和防病毒公司有用。它也应该吸引学术界以及任何参与数字数据分析的个人或公司，无论是软件开发人员、分析师、数学家还是休闲解谜者。

        


                                
        目标
        
                                
        希望通过通过 GitHub 发布 CyberChef，可以添加贡献，这些贡献可以推广到该工具的未来版本中。

        


                                
        
                                
        CyberChef 中有 数百个 有用的操作，适用于任何从事与 Internet 相关工作的人员，无论您只是想将时间戳转换为不同的格式、解压缩 gzip 压缩的数据、创建 SHA3 哈希还是解析 X.509 证书以找出颁发者。
        
                                
        它是网络瑞士军刀。

            下载 CyberChef
            
                    
            
                    
            
                        
            
                            CyberChef 完全在您的浏览器中运行，没有服务器端组件，这意味着您的输入数据和 Recipe 配置不会发送到任何地方，无论您使用的是 CyberChef 的实时官方版本还是下载的独立版本（假设它是未修改的）。
                        
            
                        
            
                            有三个操作会调用外部服务，即“在地图上显示”操作，它从 wikimedia.org 下载地图切片；“DNS over HTTPS”操作，它使用 Google 或 Cloudflare 服务解析 DNS 请求；以及“HTTP 请求”操作，它调用您输入的配置 URL。您可以使用浏览器的开发者控制台 (F12) 并查看“网络”选项卡来确认发出了哪些网络请求。
                        
            
                        
            
                            如果您想下载自己的 CyberChef 独立副本以在隔离网络或互联网连接受限或没有互联网连接的情况下运行，您可以从下面获取包含整个 Web 应用程序的 ZIP 文件。这可以在本地运行或托管在 Web 服务器上，无需任何配置。
                        
            
                        
            
                            请注意，独立版本永远不会自行更新，这意味着在您手动重新下载较新版本之前，它不会收到错误修复或新功能。
